# מוכן לכיתה א׳ — קונספט אפליקציה מלא

## מה האפליקציה

אפליקציה חינוכית גיימיפית לילדים בגיל גן חובה (5-6) המכסה את **כל תחומי המוכנות לכיתה א׳** דרך עולם משחקי עם דמות מלווה ומורה AI אינטראקטיבית.

**פלטפורמה:** Web PWA — עובד בדפדפן, ניתן להוסיף למסך הבית, מותאם לטאבלט ולמחשב.  
**קהל עיקרי:** ילדים 5-6, הורים, מורות גן.  
**שפה:** עברית (ממשק ותוכן).

---

## קונספט העולם — מפת הרובעים

העולם הוא **עיר מאויירת** שהדמות (הילד/ה) מטיילת בה. כל רובע מייצג תחום מוכנות אחר. הניווט הוא **לחיצה/טאפ על יעד** — הדמות הולכת לשם אוטומטית עם אנימציה.

```
             🌟 כיתה א׳ של גיבורים
                   (נפתח בסוף)
                        |
         ┌──────────────┼──────────────┐
         │              │              │
  🔤 רובע האותיות  ❤️ רובע הרגשות  🖊️ רובע היצירה
    (אקדמי)         (רגשי-חברתי)   (מוטוריקה)
```

### מנגנון נעילה/פתיחה

- כל משחק: 1-3 כוכבים ⭐
- רובע נפתח לאחר 70% מהמשחקים הקודמים
- "כיתה א׳ של גיבורים" נפתח כשכל הרובעים עם ⭐⭐ לפחות

## תחומי המוכנות ומשחקים

### 🔤 רובע האותיות — מיומנויות אקדמיות

| דלת | משחק | מיומנות |
|-----|------|---------|
| בית הצלילים | מה הצליל הראשון? | מודעות פונולוגית — צליל ראשון |
| מגדל האותיות | זיהוי אותיות + צלילים | הכרת האלפבית |
| מבצר החריזה | מה מחרוזת? | מודעות פונולוגית — חריזה |

| שוק המספרים | ספירה, כמויות 1-20 | מוכנות לחשבון |
| ספריית הסיפורים | עקיבה אחרי סיפור | הבנת שפה |

### ❤️ רובע הרגשות — מיומנויות רגשיות וחברתיות

| דלת | משחק | מיומנות |
|-----|------|---------|
| יום ראשון | התמודדות עם מסגרת חדשה | הסתגלות |

| כיכר החברים | שיתוף פעולה, תורות | מיומנויות חברתיות |
| הבית של רינה 🌟 | שיחה עם מורה AI | ביטחון עצמי |

### 🖊️ רובע היצירה — מוטוריקה

| דלת | משחק | מיומנות |
|-----|------|---------|
| סדנת הקווים | עקיבה עם אצבע | מוטוריקה עדינה |
| מגרש התנועה | קצב, קואורדינציה | מוטוריקה גסה |
| אטלייה הכתיבה | כתיבת אותיות ראשונות | אחיזת עיפרון |
| מטבח הצורות | drag & drop | תפיסה מרחבית |

## דמויות

### הילד/ה — הדמות הראשית
- בוחרים שם ומראה בסיום הרישום
- מופיעה על המפה ומטיילת בין הדלתות
- מקבלת פרסים ויזואליים (כוכבים, תגים)

### רינה — מורה AI 🎓
- דמות מאויירת שמופיעה בכל הרובעים כ"עוזרת"
- בחדר שלה — **שיחה אינטראקטיבית מלאה** עם Claude API
- מגיבה לאיך שהילד משחק (טועה הרבה → מציעה עזרה)
- מדברת עברית פשוטה, בגובה עיניים, בסבלנות

**System prompt של רינה (בסיס):**
```
אתה רינה, מורה חמה וסבלנית לילדים בגיל 5-6.
אתה מדבר עברית פשוטה וברורה.
תמיד חיובי, מעודד, ולא שיפוטי.
אתה מתמקד במוכנות לכיתה א׳ בלבד.
אם ילד שואל על נושא אחר — תחזיר אותו בחביבות למשחק.
כל תשובה שלך קצרה — מקסימום 2-3 משפטים.
```

## ארכיטקטורה — data-driven (ניתן להרחבה)

```
src/
├── content/           ← משנים רק כאן כשמוסיפים תוכן
│   ├── worlds.ts      ← הגדרת רובעים ומיקומים
│   ├── rooms.ts       ← הגדרת דלתות ומשחקים
│   └── games/
│       ├── firstSound.ts   ← מילים + תמונות + תשובות
│       ├── rhyming.ts
│       └── syllables.ts
│
└── engine/            ← לא נוגעים אחרי שבנינו
    ├── WorldMap/
    ├── GameRunner/

```

## UX ועיצוב

- **ניווט:** לחיצה על יעד → הדמות הולכת לשם (אנימציה)
- **טאבלט:** מותאם touch, כפתורים מינימום 44px
- **RTL:** עברית ימין לשמאל
- **אנימציות:** Framer Motion — כניסות, מעברים, חגיגות הצלחה
- **קול:** Web Speech API (TTS לקריינות) + אפקטי קול
- **ללא גלילה:** כל מסך = viewport מלא (כמו Genially)

## רעיונות עתידיים (מחוץ לסקופ הראשוני)

### אפליקציות נפרדות שניתן לבנות אחר כך

**ארמון הכפל** (ב׳-ד׳)
- Memory Palace — כל חדר = כפולה אחת
- סיפורים ויזואליים מגוחכים לזיכרון
- "7×8=56: שבעה פינגווינים + שמונה בלונים = 56 כוכבים"

**חשבון + רגשות** (א׳-ב׳)
- מד רגשות לפני כל שיעור
- קושי משחק מתאים את עצמו לפי מצב רגשי
- מלמד מטא-קוגניציה

**מסלול הכתיבה** (גן-א׳)
- עקיבה עם אצבע → אימון מוטוריקה עדינה
- מדידת איכות כתיבה (לחץ, מהירות, כיוון)
- אבחון + תרגול מוטוריקה עדינה לפני כיתה א׳

**הדרך שלי — הוראה מתקנת** (א׳-ד׳)
- Multi-sensory לאותיות מבלבלות (ב/כ, ד/ר)
- מעקב דפוסי טעויות → אבחון אוטומטי
- מותאם לדיסלקציה / דיסקלקוליה
